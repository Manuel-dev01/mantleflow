# /web — MantleFlow frontend (Phase 1)

Next.js (App Router) + TypeScript + Tailwind + shadcn/ui. Deploys to Vercel. Hosts the query UI and
the HTTP API routes that call `@mantleflow/agent`. Every displayed datum renders its `SourceReceipt`
(source + timestamp + fact/estimate/assumption). Scaffolded in Phase 1 with the MI4 vertical slice.
